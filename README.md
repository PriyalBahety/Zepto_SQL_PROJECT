# Zepto SQL Project

A PostgreSQL-based data analysis project using Zepto grocery inventory data. This project focuses on importing CSV data into a database and using SQL queries to extract useful business insights such as pricing trends, discounts, stock availability, and category-wise analysis.

## Features

* Import CSV data into PostgreSQL
* Clean and manage raw product data
* Analyze discounts and selling prices
* Check stock availability
* Category-wise product insights
* Practice real-world SQL queries

## Dataset Columns

* Category
* Product Name
* MRP
* Discount Percent
* Available Quantity
* Discounted Selling Price
* Weight
* Out of Stock
* Quantity

## SQL Concepts Used

* SELECT, WHERE, ORDER BY
* GROUP BY, HAVING
* Aggregate Functions
* CASE Statements
* Subqueries
* Views
* COPY / \copy for CSV Import

## Tools Used

* PostgreSQL
* pgAdmin
* CSV Dataset

## Example Insights

* Highest discounted products
* Average price by category
* Out of stock products
* Total products in each category

## How to Run

1. Create database in PostgreSQL
2. Create table using schema
3. Import CSV file using COPY command
4. Run SQL queries for analysis
